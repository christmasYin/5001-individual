The programming language is Python.

Packages including pandas, numpy, sklearn, time, datetime and pickle are required to run the code.

Put the training data, testing data and sample submission in the same path namely 'train.csv', 'test.csv' and 'sample submission.csv' respectively.

To reproduce the prediction results, you are supposed to create a empty file named 'save', since the trained model will be saved in this file and it will be invoked from this file when testing.
